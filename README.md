# News Scraper

> A web scraper of [SD Times](https://sdtimes.com/) where users can view headlines and leave comments

**Click [here](https://young-shore-59428.herokuapp.com/) to view the deployed site on Heroku**

## Tools

* Node, Express
* MongoDB, Mongoose
* Cheerio, Request
* Handlebars
* Mocha, Chai, Chai-http
* ESLint
